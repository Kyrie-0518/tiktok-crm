# 跨境电商概述

## 什么是跨境电商

跨境电商（Cross-Border E-Commerce）是指分属不同关境的交易主体，通过电子商务平台达成交易、进行支付结算，并通过跨境物流送达商品、完成交易的一种国际商业活动。

## 主要模式

| 模式 | 说明 | 代表平台 |
|------|------|----------|
| **B2B** | 企业对企业批发 | 阿里巴巴国际站、敦煌网 |
| **B2C** | 企业对消费者零售 | 亚马逊、速卖通、TikTok Shop |
| **C2C** | 个人对个人 | eBay、闲鱼国际 |
| **D2C** | 品牌直营 | Shopify独立站、品牌官网 |
| **社交电商** | 通过社交媒体成交 | TikTok Shop、Instagram Shopping |

## 中国跨境电商发展阶段

1. **1.0 时代（1999-2012）**：以阿里巴巴国际站为代表的信息撮合平台
2. **2.0 时代（2013-2019）**：亚马逊全球开店、速卖通等交易平台崛起
3. **3.0 时代（2020-至今）**：TikTok Shop领衔的社交电商+直播电商时代
4. **4.0 趋势**：AI驱动、全托管模式、本土化深耕

## 全球市场规模

- 2023年全球跨境电商市场规模约 **2.8万亿美元**
- 东南亚电商市场增速全球第一，年增长率 **15-20%**
- TikTok Shop 2023年东南亚GMV超 **150亿美元**

## 核心参与者

- **卖家**：品牌商、工厂、贸易商、个人创业者
- **平台**：提供交易基础设施与服务
- **服务商**：物流、支付、ERP、营销、合规等
- **消费者**：全球终端买家

## 产业链全景图

```
生产制造 → 品牌运营 → 平台开店 → 营销引流 → 订单成交 → 跨境物流 → 海外仓储 → 尾程配送 → 售后服务
                                              ↓
                                         支付结算/换汇
```
