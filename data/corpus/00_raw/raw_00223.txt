# 跨境新手入门指南

## 一、我适合做跨境电商吗？

### ✅ 适合做的人
- 有供应链资源（工厂/亲戚做批发）
- 有电商经验（淘宝/拼多多卖家转型）
- 执行力强、愿意每天花4小时+学习
- 启动资金 $2000-5000
- 对海外市场有兴趣

### ❌ 暂时不适合的人
- 启动资金不足 $500
- 没有时间研究市场（每天 < 1小时）
- 期望立竿见影（期望1个月回本）
- 无法接受初期亏损

## 二、从0到1的12周计划

### 第1-2周：学习调研
- [ ] 看10个TikTok跨境卖家账号
- [ ] 学习平台基础规则
- [ ] 确定目标市场（建议从东南亚1个国家开始）
- [ ] 浏览1688/拼多多，了解供应链

### 第3-4周：注册开店
- [ ] 准备营业执照
- [ ] 注册TikTok Shop / Shopee 账号
- [ ] 完成店铺基础设置
- [ ] 了解后台操作

### 第5-6周：选品
- [ ] 使用Kalodata/Tabcut分析热卖品
- [ ] 找到3-5个潜在产品
- [ ] 1688采购样品
- [ ] 评估产品质量和成本
- [ ] 确定首发1-2个产品

### 第7-8周：上架与基础运营
- [ ] 制作产品主图/视频
- [ ] 写Listing文案
- [ ] 发布商品
- [ ] 学习后台数据分析
- [ ] 开始每天发布2-3条短视频

### 第9-10周：出单测试
- [ ] 观察自然流量数据
- [ ] 如果无流量：优化视频内容
- [ ] 如果有点击无转化：检查价格和详情页
- [ ] 如果有转化：准备补货

### 第11-12周：规模扩展
- [ ] 追加1-2个新品
- [ ] 尝试小预算广告投放（$20/天起）
- [ ] 联系3-5个达人合作
- [ ] 尝试首次直播

## 三、新手必知的5个数字

| 指标 | 含义 | 重要性 |
|------|------|--------|
| **3倍** | 售价至少为采购价3倍 | ⭐⭐⭐⭐⭐ |
| **30%** | 毛利率健康线 | ⭐⭐⭐⭐⭐ |
| **30单/天** | 值得入海外仓的拐点 | ⭐⭐⭐⭐ |
| **15%** | 退货率警戒线 | ⭐⭐⭐⭐ |
| **7天** | 视频冷启动观察期 | ⭐⭐⭐ |

## 四、启动资金预算（以东南亚最小单位为例）

| 项目 | 金额 | 备注 |
|------|------|------|
| 营业执照 | ¥500-2000 | 一次性 |
| 首批采购 | ¥2000-5000 | 产品+样品 |
| 物流首单 | ¥500-1500 | 直邮成本 |
| 内容制作 | ¥0-1000 | 手机拍摄基本0成本 |
| 小预算广告 | ¥500-2000 | 测试用 |
| 其他（工具/认证） | ¥500-2000 | |
| **合计** | **¥4000-13500** | |

## 五、新手最容易遇到的问题和解决

| 问题 | 解决方案 |
|------|----------|
| 视频0播放 | 检查账号权重、换钩子、检查画面质量 |
| 有点击无购买 | 检查价格、评价、详情页描述 |
| 物流太慢被差评 | 切换物流渠道、考虑预付海外仓 |
| 收到侵权警告 | 立即下架、排查产品、申诉 |
| 钱被平台冻结 | 联系平台客服、提供合规证明 |
