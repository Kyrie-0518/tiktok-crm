# Listing视觉设计指南

## 一、主图设计黄金法则

### 第1张主图：白底正面图
- 纯白背景（#FFFFFF 或 #F5F5F5）
- 产品占画面 80%
- 正面展示、清晰无模糊
- 无Logo水印挡住产品

### 第2张主图：功能卖点图
- 标注2-3个核心卖点
- 文字简洁（每个卖点少于5个字）
- 箭头指向对应位置
- 配色不花哨（1-2种辅助色）

### 第3张主图：场景使用图
- 真人使用/穿着效果
- 生活化场景（客厅/办公室/户外）
- 自然光线、不过度P图
- 让人能想象自己使用时效果

### 第4张主图：尺寸/对比图
- 实物大小对比（人手/硬币/手机）
- 多种颜色/款式排列
- 尺寸标注清晰
- 比例尺准确

### 第5张主图：细节展示
- 材质纹理特写
- 做工精细度
- 独特设计亮点
- 包装/附件展示

## 二、短视频内容设计

### 15秒产品视频模板
```
0-3秒：钩子（惊讶/好奇/痛点）
3-8秒：产品展示（多角度+使用场景）
8-12秒：效果对比（before/after）
12-15秒：行动号召（限时/库存/购买引导）
```

### 视频类型比例
| 类型 | 时长 | 发布频率 | 目的 |
|------|------|----------|------|
| 产品展示 | 15-20秒 | 2条/天 | 种草 |
| 开箱测评 | 20-30秒 | 1条/天 | 信任 |
| 使用教程 | 30-60秒 | 3条/周 | 教育 |
| 对比测试 | 15-25秒 | 2条/周 | 竞争 |
| UGC混剪 | 10-15秒 | 1条/周 | 社交证明 |

## 三、详情页设计规范

### 详情页结构
```
┌─────────────────────────┐
│ 主图轮播                │
├─────────────────────────┤
│ 价格 + 促销信息          │
├─────────────────────────┤
│ 规格选择(颜色/尺寸/款式)  │
├─────────────────────────┤
│ 核心卖点(3-5条)          │
├─────────────────────────┤
│ 产品详情(图文+表格)      │
│ - 材质/尺寸/重量         │
│ - 功能说明               │
│ - 使用教程               │
├─────────────────────────┤
│ 好评精选(social proof)   │
├─────────────────────────┤
│ 售后保障/退换政策         │
└─────────────────────────┘
```

### 详情图设计要求
- 宽度：750-1200px
- 格式：JPG/PNG/Webp
- 字体：清晰可读，中英文分开排版
- 配色：品牌色系，不超过3色
- 留白：每张图只讲一个点

## 四、文案撰写公式

### 标题公式
```
核心关键词 + 属性词 + 场景词 + 利益点
```

**优质标题示例**：
- `高端商务真皮公文包|大容量14寸笔记本包|出差旅行|送礼优选` ✅
- `夏季轻薄速干T恤男|纯棉宽松圆领|运动健身百搭|不起球不掉色` ✅
- `包包` ❌

### 五点描述（Bullet Points）
```
1. 【材质】100%纯棉面料，亲肤透气不闷汗
2. 【设计】3D立体剪裁，显瘦遮肉修饰身形
3. 【功能】大容量+多口袋，手机/钥匙/钱包统统装下
4. 【场景】通勤/约会/度假，一条搞定所有场合
5. 【保障】30天无理由退换，质量问题包邮退
```

## 五、色彩心理学应用

| 颜色 | 心理效应 | 适用品类 |
|------|----------|----------|
| 🔴 红 | 紧迫感、促销 | 大促/清仓/限时 |
| 🔵 蓝 | 信任、专业 | 科技/商务/金融 |
| 🟢 绿 | 健康、自然 | 食品/母婴/环保 |
| 🟡 黄 | 年轻、活力 | 潮牌/运动/时尚 |
| ⚫ 黑 | 高端、奢华 | 奢侈品/高端3C |
| 📱 粉 | 甜美、女性 | 女性/美妆 |
