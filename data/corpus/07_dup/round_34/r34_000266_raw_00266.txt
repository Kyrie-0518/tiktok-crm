# 合规与风险防控

## 一、知识产权

### 三种主要知识产权
| 类型 | 保护对象 | 侵权形式 | 高风险行为 |
|------|----------|----------|------------|
| **商标** | 品牌名称/LOGO | 假冒/仿冒 | 使用他人品牌名 |
| **专利** | 发明/外观设计 | 仿制专利产品 | 仿爆款设计 |
| **版权** | 图片/视频/文案 | 盗图/洗稿 | 直接使用他人素材 |

### 防侵权 CHECKLIST
- [ ] 产品外观/设计无专利冲突（可查WIPO/各国专利局）
- [ ] 品牌名/LOGO无商标冲突
- [ ] 图片/视频素材为原创或已授权
- [ ] 产品文案为原创（非直接翻译竞品）
- [ ] 达人合作内容有授权协议

## 二、税务合规

### 东南亚各国税务

| 国家 | 增值税率 | 起征点 | 特殊税种 |
|------|----------|--------|----------|
| 印尼 | 11% | - | 进口税7.5%-20% |
| 泰国 | 7% | 1500THB | - |
| 越南 | 10% | 1M VND | - |
| 马来西亚 | 6%(SST) | - | 进口关税 |
| 菲律宾 | 12% | 10000PHP | - |

### 欧盟（英国/欧洲）
- IOSS系统：150欧元以下商品统一报税
- 增值税率：17%-27%（因国家而异）
- EPR（生产者责任延伸）：包装/电池/电子废弃物

### 美国
- 无联邦销售税
- 各州销售税不同（0%-10%）
- 达到一定额度需注册各州税号

## 三、平台合规

### TikTok Shop 红线
1. ❌ **延迟发货**：发货率低于 95% → 限流
2. ❌ **虚假发货**：空包/错包 → 扣分封店
3. ❌ **引导站外交易**：电话/微信/WhatsApp → 永久封禁
4. ❌ **虚假宣传**：夸大功效 → 商品下架+罚款
5. ❌ **价格欺诈**：虚构原价 → 扣分
6. ❌ **刷单刷评**：虚假交易 → 封店

### 类目准入要求
- **美妆护肤**：需要品牌授权/检验报告
- **食品/保健品**：需要当地食药监认证
- **电子产品**：需要安全认证（CE/FCC/SNI等）
- **儿童用品**：需要安全检测报告

## 四、风险防控矩阵

| 风险类型 | 概率 | 影响 | 防控措施 |
|----------|------|------|----------|
| 账号封禁 | 中 | 致命 | 多账号矩阵、合规运营 |
| 资金冻结 | 中 | 严重 | 分散回款、第三方支付 |
| 侵权诉讼 | 低 | 致命 | 知识产权排查 |
| 汇率波动 | 高 | 中等 | 多币种账户、锁汇 |
| 物流延误 | 中 | 中等 | 多物流渠道备选 |
| 退货潮 | 中 | 中等 | 质量管控、描述真实 |

## 五、危机应对 SOP

```
发现问题 → 2小时内响应
    │
    ├── 平台违规 → 自查 + 整改 + 申诉
    ├── 客户投诉 → 道歉 + 赔偿 + 流程改进
    ├── 差评爆发 → 逐一跟进 + 改进产品
    └── 物流延误 → 主动通知 + 补偿方案
```
