# 广告素材创意库

## 一、TikTok广告素材公式

### 爆款素材结构
```
3秒钩子 → 10秒展示 → 5秒对比 → CTA
```
每条素材只能有一个核心信息点。

### 已验证的钩子模板
1. **对比型**："买之前 vs 买之后"、"左边便宜货 vs 右边我家"
2. **惊讶型**："这个价格你敢信？"、"这真的是同一个产品？"
3. **痛点型**："你家的XX是不是也这样？"
4. **教程型**："学会这招，省下XXX元"
5. **开箱型**："刚收到的快递，赶紧拆开看看"

## 二、素材类型分类

### 真人出镜类
- 使用示范（最佳转化）
- 开箱测评（建立信任）
- 对比测试（优胜展示）
- 素人街访（真实感）

### 产品展示类
- 微距特写（质感）
- 多角度展示
- 功能演示
- 包装展示

### 场景类
- 使用场景还原
- 生活方式展示
- UI/数字产品演示

## 三、A/B测试方法论

### 测试变量
| 变量 | 示例A | 示例B |
|------|-------|-------|
| 钩子 | 惊讶型 | 痛点型 |
| 展示角度 | 正面 | 侧面 |
| 背景 | 纯色 | 生活场景 |
| BGM | 热门A | 热门B |
| 价格位置 | 开头 | 结尾 |
| CTA | "限时优惠" | "库存不多" |

### 测试规则
- 一次只改变一个变量
- 每个素材跑最少$50或2000展示
- 统计学显著才做判断
- 胜出的素材做Templatize（模板化）

## 四、素材疲劳与更新

### 疲劳信号
- CTR连续3天下降
- CPM在上升但转化没涨
- ROAS下降超过20%

### 更新策略
```
每周：更新1-2个新视频素材
每2周：更换一轮广告组内的素材
每月：大更新（全新创意方向）
```

### 素材复用技巧
```
同一个产品 → 
  素材A: 强调功能
  素材B: 强调价格
  素材C: 强调品质
  素材D: 强调使用场景
```

## 五、不同类型产品素材指南

### 服装类
- 上身效果（多体型展示）
- 穿搭组合
- 面料特写
- before/after

### 3C配件类
- 安装教程
- 对比效果
- 防水/防摔测试
- 充电/续航展示

### 美妆类
- 半脸对比（最有说服力）
- 24小时持妆测试
- 色号试色
- 使用教程

### 家居类
- 空间改造before/after
- 收纳功能展示
- 节省空间对比
- 清洁效果
