# 新手从0到1实操手册

## 一、开店前准备（第1-3天）

### Day 1：准备工作
- [ ] 准备营业执照（个体工商户即可）
- [ ] 准备法人身份证
- [ ] 准备手机号（未注册过平台）
- [ ] 准备邮箱

### Day 2：平台选择
选择你的第一个平台：
```
TikTok Shop（推荐新手）：
✅ 流量巨大
✅ 内容驱动，不依赖广告
✅ 入驻门槛低
❌ 需要做视频/直播

Shopee：
✅ 体系成熟
✅ 新手扶持多
❌ 竞争激烈
```

### Day 3：注册店铺
1. 访问官方商家中心
2. 填写企业信息
3. 上传证件
4. 等待审核（1-3个工作日）

## 二、选品与上架（第4-7天）

### Day 4-5：选品
1. 打开1688，搜索"跨境[品类]"
2. 看销量排名前100
3. 筛选毛利润>3倍的产品
4. 下单购买样品（3-5个）

### Day 6-7：上架
1. 拍摄产品主图（白底+场景+细节）
2. 写标题（核心词+属性+场景）
3. 填写产品参数
4. 设置价格（成本×3以上）
5. 设置库存（首次10-20件即可）

## 三、首条视频与第一单（第8-14天）

### Day 8-10：视频制作
```
拍摄清单：
1. 找一个光线好的地方
2. 手机横屏拍摄、画面稳定
3. 前3秒必须有钩子
4. 视频时长15-25秒
5. 用CapCut剪辑+加字幕
6. 选平台热门BGM
```

### Day 11-14：等待第一单
- 每天发布2-3条视频
- 回复评论和私信
- 设置自动回复
- 如果没有播放→优化视频内容

### 第一单来了！
1. 核实订单信息
2. 24小时内发货
3. 如实填写物流单号
4. 跟踪物流状态
5. 收货后跟进好评

## 四、优化与扩展（第15-30天）

### 数据复盘
```
每天花10分钟看数据：
- 视频播放量趋势
- 访客数和转化率
- 哪个产品最受欢迎
```

### 优化方向
- 播放量低 → 换视频钩子
- 点击率低 → 优化主图+标题
- 转化率低 → 检查价格+详情页
- 没复购 → 包裹放感谢卡+折扣码

## 五、新手30天目标

| 周数 | 目标 | 关键动作 |
|------|------|----------|
| 第1周 | 店铺开起来 | 注册+选品 |
| 第2周 | 出第一单 | 发视频+优化 |
| 第3周 | 日出5单 | 追加新品+优化视频 |
| 第4周 | 日出10单 | 找达人+小预算广告 |
