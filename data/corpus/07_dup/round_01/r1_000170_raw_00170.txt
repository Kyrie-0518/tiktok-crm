# 进出口政策与海关实务

## 一、中国出口政策框架

### 跨境电商出口主要模式
| 模式 | 代码 | 说明 | 退税 |
|------|------|------|------|
| 9610 | 跨境电商B2C直邮 | 清单核放、汇总申报 | 可退税 |
| 9710 | 跨境电商B2B直接出口 | 企业对企业 | 可退税 |
| 9810 | 跨境电商出口海外仓 | 先出口至海外仓再销售 | 可退税 |
| 0110 | 一般贸易 | 传统B2B | 可退税 |
| 1039 | 市场采购贸易 | 试点区域 | 简化 |

### 出口退税
- 基本税率：大部分商品退税率为 13%（增值税全退）
- 部分品类：9%、0%（资源类、限制类）
- 条件：取得增值税专用发票 + 已报关出口 + 收汇
- 时间：申报后 1-3 个月到账

## 二、主要目的国进口政策

### 东南亚

#### 印尼
- **关税**：0%-20%（按HS编码）
- **增值税**：11%（PPN）
- **所得税**：7.5%-20%（PPh 22进口所得税）
- **特殊要求**：
  - 化妆品需 BPOM 认证
  - 电子产品需 SNI 认证
  - 食品需 BPOM + 清真认证（Halal）
- **新政**：2023年限制 100美元以下跨境商品入市

#### 泰国
- **关税**：0%-30%
- **增值税**：7%
- **起征点**：1500泰铢以下免关税
- **特殊要求**：化妆品需 FDA 认证、电子产品需 NTC

#### 越南
- **关税**：0%-35%
- **增值税**：10%
- **起征点**：100万越南盾以下免关税
- **特殊要求**：部分品类需进口许可

#### 马来西亚
- **关税**：0%-40%（多为 0%-20%）
- **SST**：6%（销售和服务税）
- **特殊要求**：电子产品需 SIRIM 认证

#### 菲律宾
- **关税**：0%-30%
- **增值税**：12%
- **起征点**：10000比索以下免税
- **特殊要求**：电子产品需 NTC、食品需 FDA

### 欧美

#### 美国
- **关税**：0%-25%（301条款对中国部分商品加征）
- **无联邦销售税**（各州自定）
- **新政**：2025年取消T86小额免税（800美元以下）
- **特殊**：FCC认证（电子）、CPC（儿童产品）

#### 欧盟
- **关税**：0%-17%
- **增值税**：17%-27%（因国而异）
- **新政**：2021年起取消22欧元免税；IOSS一站式申报
- **特殊**：CE认证、GPSR（通用产品安全法规）、EPR

## 三、报关流程

### 简易B2C报关（9610）
```
订单生成 → 打包 → 生成运单 → 清单申报 → 查验 → 放行 → 出口
```

### 海外仓出口（9810）
```
采购备货 → 一般贸易出口 → 海运/空运 → 目的国清关 → 入仓 → 销售 → 尾程配送
```

## 四、各国禁限售品概览

| 国家 | 禁售/限制品 |
|------|------------|
| 印尼 | 100美元以下跨境商品、化妆品（无BPOM） |
| 泰国 | 电子烟、赌博相关 |
| 越南 | 二手服装、部分电子产品 |
| 马来 | 酒精、烟草、武器 |
| 美国 | 仿品、儿童产品（无CPC） |
| 欧盟 | 无CE认证电子品、无GPSR合规品 |

## 五、应对海关查验

### 高频查验原因
1. 申报价值异常（过高或过低）
2. 品类描述模糊
3. HS编码不匹配
4. 被列入重点监控名单

### 应对策略
- 真实申报、不低报
- 保留采购证明和交易记录
- 提前了解目的国合规要求
- 使用合规的物流报关代理
