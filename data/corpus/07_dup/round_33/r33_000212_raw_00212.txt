# 跨境电商知识产权保护

## 一、三种知识产权类型

### 商标（Trademark）
- **保护**：品牌名称、LOGO、Slogan
- **侵权风险**：使用他人商标、相似商标
- **保护期限**：10年（可续期）
- **查询**：各国商标局官网

### 专利（Patent）
- **外观专利**：产品外观设计
- **发明专利**：技术方案/功能
- **侵权风险**：销售专利产品仿制品
- **保护期限**：外观10-15年、发明20年

### 版权（Copyright）
- **保护**：图片、视频、文案、设计
- **侵权风险**：盗图、洗稿、抄文案
- **保护期限**：作者终身+50-70年

## 二、常见侵权场景

| 场景 | 侵权类型 | 预防 |
|------|----------|------|
| 使用竞品图片 | 版权 | 自己拍摄 |
| 品牌名含大牌词 | 商标 | 查商标数据库 |
| 仿制爆款外观 | 专利 | 改设计 |
| 翻译竞品文案 | 版权 | 原创文案 |
| 视频用未授权BGM | 版权 | 用平台音乐库 |
| LOGO相似知名品牌 | 商标 | 原创LOGO |

## 三、防侵权CHECKLIST

### 新品上线前
- [ ] 产品外观是否与已专利设计相似？
- [ ] 品牌名是否包含他人注册商标？
- [ ] 产品图片是否为原创或已授权？
- [ ] 文案是否为原创（非翻译竞品）？
- [ ] 视频BGM是否来自平台音乐库？

### 查询渠道
- **中国商标**：中国商标网 sbj.cnipa.gov.cn
- **美国商标/专利**：USPTO www.uspto.gov
- **欧盟商标/外观**：EUIPO euipo.europa.eu
- **WIPO全球查**：www.wipo.int

## 四、被投诉侵权怎么办

### 立即行动
```
1. 下架被投诉产品（立即！）
2. 分析投诉内容
3. 判断是否真的侵权
   ├── 真的侵权 → 联系投诉方和解
   │              → 承诺不再侵权
   │              → 协商赔偿（如需）
   └── 不侵权 → 提交申诉材料
                → 提供原创证明
```

### 申诉材料准备
- 产品原创设计证明
- 品牌注册证书
- 采购授权书
- 对比分析（证明不相似）

## 五、主动保护自己的IP

### 海外商标注册
| 注册途径 | 覆盖范围 | 费用 |
|----------|----------|------|
| 单一国家注册 | 1个国家 | $300-2000 |
| 马德里体系 | 多国 | $2000+ |
| 欧盟商标 | 欧盟27国 | €850起 |
| 美国商标 | 美国 | $250-350（官费） |

### 优先注册
- 品牌名（核心市场）
- LOGO
- 核心产品外观（如有独特性）
