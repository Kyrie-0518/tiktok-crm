# 产品生命周期管理

## 一、产品生命周期四阶段

```
导入期 → 成长期 → 成熟期 → 衰退期
         ↑                    │
         └──── 迭代升级 ←─────┘
```

### 阶段1：导入期（0-4周）
| 特征 | 策略 |
|------|------|
| 销量低 | 内容高频测试、达人寄样 |
| 成本高 | 小预算广告测试 |
| 无利润 | 允许亏损验证 |
| 核心目标 | 验证产品-市场匹配 |

**判断标准**：
- 连续7天自然出单 → 可进入成长期
- 7天0单 → 换品或大改

### 阶段2：成长期（1-3个月）
| 特征 | 策略 |
|------|------|
| 销量快速增长 | 追加库存 + 海外仓 |
| 利润转正 | 优化成本结构 |
| 竞争上来了 | 建立评价壁垒 |

**核心动作**：
- 加大广告预算(逐步)
- 扩大达人合作
- 增加视频产出频率
- 关注竞品动态

### 阶段3：成熟期（3-12个月）
| 特征 | 策略 |
|------|------|
| 销量稳定 | 维持ROI |
| 利润可观 | 降本提效 |
| 多种产品 | 产品矩阵 |

**核心动作**：
- 稳定广告投入
- UGC内容维护
- 延伸产品线（颜色/规格/套装）
- 考虑独立站承接

### 阶段4：衰退期
| 特征 | 策略 |
|------|------|
| 销量下滑 | 逐步清仓 |
| 利润压缩 | 探索迭代升级 |

**核心动作**：
- 是否还能迭代（换配色/升级功能）
- 如果不值得迭代 → 清库存出局
- 将资源转移到新品

## 二、SKU管理策略

### 爆旺平滞分类
| 类型 | 定义 | 策略 |
|------|------|------|
| 爆款 | TOP 10% SKU | 重点投入（库存/广告） |
| 旺款 | TOP 10-30% | 维护、小幅优化 |
| 平款 | 30-80% | 观察、不主动加量 |
| 滞款 | 后20% | 清仓/下架 |

### 定期清理机制
- 每周检查"0销量超过14天"的SKU
- 每月砍掉底部10%的滞销品
- 每季度评估品类结构

## 三、产品矩阵

### 波士顿矩阵
```
        市场增长率
             ↑
    明星产品  │  问题产品
  (加大投入) │ (选择性投入)
             │
  ───────────┼───────────→ 市场份额
             │
    金牛产品  │  瘦狗产品
  (收割利润) │ (清仓退出)
             ↓
```

### 健康产品结构
- 1-2个爆款（占GMV 50%）
- 3-5个旺款（占GMV 30%）
- 5-10个平款（占GMV 15%）
- 少量新品测试（占GMV 5%）

## 四、产品迭代升级

### 升级路径
```
V1.0 基础款 → V2.0 小改款(颜色/包装) → V3.0 重大改款(功能/品质) → V4.0 全新一代
```

### 升级时机
- 销量连续2周下降
- 差评集中在某个缺陷
- 竞品有了更好版本
- 产品成本有了下降空间

## 关键词

产品生命周期、爆旺平滞、产品矩阵、波士顿矩阵、SKU管理、迭代升级
