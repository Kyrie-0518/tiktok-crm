# 多仓布局与库存网络

## 一、仓储布局策略

### 为什么要多仓
- 离消费者更近 → 更快的配送
- 分散风险 → 单仓出问题不影响全局
- 降低尾程成本 → 本地配送更便宜

### 仓储布局决策
```
中心仓（中国/中转）
    │
    ├→ 东南亚区域仓（泰国/马来）
    │     ├→ 印尼前置仓
    │     └→ 越南前置仓
    │
    └→ 欧美区域仓（美国/德国）
          ├→ 美东仓
          └→ 美西仓
```

## 二、何时扩展仓储

### 单仓扩容触发点
| 指标 | 触发值 | 行动 |
|------|--------|------|
| 仓库利用率 | > 85% | 扩容或分流 |
| 某市场日单量 | > 50单 | 该市场单独设仓 |
| 配送超时率 | > 10% | 检查仓储位置 |
| 尾程成本涨幅 | > 15% | 考虑多仓分摊 |

### 分仓决策矩阵
| 条件 | 不分仓 | 分仓 |
|------|--------|------|
| 日单量 | < 100 | > 300 |
| 市场数 | 1-2个 | 3个+ |
| 配送时效要求 | < 5天可接受 | 必须1-3天 |
| 资金实力 | 紧张 | 充裕 |

## 三、库存调配

### 如何分配库存到各仓
```
总库存 1000件
  │
  ├→ A仓（核心市场）: 500件（占60%销量）
  ├→ B仓（增长市场）: 300件（占30%销量）
  └→ C仓（测试市场）: 200件（占10%销量）
```

### 仓间调拨
- A仓库存不足 + B仓库存过剩 → 调拨
- 调拨成本一般 $0.5-2/件
- 比断货的损失小得多

## 四、库存预警系统

### 预警线设置
```
红色预警（紧急）：库存 < 3天销量
黄色预警（注意）：库存 < 7天销量
绿色安全：库存 > 14天销量
```

### 自动补货建议
```
当库存降到黄色预警时：
→ 如果头程补货可在7天内到达 → 自动下单
→ 如果头程需要15天 → 立即下单
→ 如果来不及 → 从其他仓调拨
```

## 五、仓储成本优化

### 仓储费用构成
| 费用 | 价格参考 | 优化方式 |
|------|----------|----------|
| 仓租费 | $0.5-3/件/月 | 加快周转、清滞销品 |
| 操作费 | $0.5-2/单 | 标准化包装、减少操作 |
| 增值服务 | 按需 | 自己能做的别外包 |

### 滞销品清理
```
库存超过90天 → 评估是否打折清仓
库存超过180天 → 必须清（仓储费已超过货值）
```
