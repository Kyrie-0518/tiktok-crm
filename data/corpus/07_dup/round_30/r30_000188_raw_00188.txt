# 营销推广方法论

## 一、TikTok 内容营销

### 爆款视频公式
```
钩子（3秒）→ 痛点/好奇（5秒）→ 产品展示（10秒）→ 效果对比（5秒）→ CTA（2秒）
```

### 视频类型矩阵
| 类型 | 占比 | 目的 | 时长 |
|------|------|------|------|
| 产品展示 | 30% | 介绍功能 | 15-25秒 |
| 使用教程 | 20% | 建立信任 | 30-60秒 |
| 开箱测评 | 15% | 真实感 | 20-30秒 |
| 对比测试 | 15% | 凸显优势 | 15-25秒 |
| 用户反馈 | 10% | 社交证明 | 10-20秒 |
| 幕后花絮 | 10% | 人设打造 | 15-30秒 |

### 发布策略
- 每日发布：2-3条
- 最佳时段：当地 12:00-14:00 / 19:00-22:00
- 固定模式：每周特定时间发布固定系列
- 标签策略：3-5个精准标签 + 1-2个热门标签

## 二、直播带货

### 直播排品策略

| 顺序 | 品类型 | 目的 | 利润率 |
|------|--------|------|--------|
| 开播0-10分 | 福利款（1-2个） | 拉停留 | 微利/平出 |
| 10-30分 | 利润款（2-3个） | 赚利润 | 高利润 |
| 30-45分 | 爆款（1-2个） | 冲GMV | 中利润 |
| 45-55分 | 引流款 | 拉在线人数 | 微利 |
| 55-60分 | 返场爆款 | 收尾冲量 | 中利润 |

### 直播话术结构
```
痛点引入 → 产品展示 → 信任构建 → 价格锚定 → 限时限量 → 下单引导
```

### 直播指标
- **场观**：直播间总观看人数
- **在线人数**：实时观看数，目标稳定在50+
- **GPM**：千次观看成交额，目标 > 500
- **转化率**：下单/观看，目标 > 2%

## 三、达人营销（Influencer Marketing）

### 达人分层
| 层级 | 粉丝量 | 合作方式 | 适合场景 |
|------|--------|----------|----------|
| 头部 | 100万+ | 坑位费+佣金 | 品牌声量 |
| 腰部 | 10-100万 | 坑位费或纯佣 | 品效合一 |
| 尾部 | 1-10万 | 纯佣金 | 铺量测试 |
| 素人 | <1万 | 寄样置换 | 内容矩阵 |

### 合作流程
```
达人筛选 → 寄样 → 内容确认 → 发布 → 数据复盘 → 续约
```

### 达人评估指标
- 粉丝匹配度（目标市场、年龄段、兴趣）
- 近期视频平均播放量
- 互动率（点赞+评论+分享 / 播放量）
- 过往带货转化数据
- 内容风格是否与品牌调性一致

## 四、社媒矩阵

- **TikTok**：短视频+直播+橱窗
- **Facebook**：社群运营+广告
- **Instagram**：图文+Reels+购物标签
- **YouTube**：深度测评
- **WhatsApp/Line**：私域沉淀

内容营销、直播带货、达人营销、视频脚本、排品策略、KOL合作
