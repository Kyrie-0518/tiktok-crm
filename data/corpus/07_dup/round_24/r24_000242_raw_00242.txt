# 客户服务与售后管理

## 一、客服响应标准

### 响应时间
| 渠道 | 工作日 | 周末/节假日 |
|------|--------|------------|
| 平台聊天 | < 2小时 | < 4小时 |
| 邮件 | < 24小时 | < 48小时 |
| WhatsApp | < 1小时 | < 3小时 |
| 社交媒体评论 | < 4小时 | < 8小时 |

### 自动回复模板

**欢迎语**：
```
Hi! Thank you for reaching out! 
We're here to help 😊 
Our working hours: Mon-Fri 9AM-6PM (GMT+8)
We'll reply within 2 hours.
```

**常见问题快捷回复**：
```
【物流查询】
Your order has been shipped on [date]. 
Tracking number: [xxx]
Track here: [link]

【退款申请】
We're sorry to hear that. Please tell us:
1. What's wrong with the product?
2. Can you send a photo?
We'll process your refund ASAP!
```

## 二、差评处理SOP

### 收到差评后的步骤
```
1. 24小时内回复（公开回复，展示专业态度）
2. 私下联系客户 → 道歉 + 了解详情
3. 提出解决方案：
   - A. 全额退款（不退货）
   - B. 补发新品
   - C. 优惠券补偿
4. 客户满意后礼貌请求修改评价
5. 内部复盘：为什么出现这个问题？
```

### 差评回复模板
```
Dear [Name],

We're truly sorry that your experience didn't meet expectations. 
We've reached out to you via message to resolve this issue. 
Please check your inbox — we want to make this right!

Best regards,
[Your Store Name]
```

## 三、退货退款流程

### 退货政策
```
退货窗口：收到商品后7-15天
退货条件：商品完好、未使用、原包装
退货运费：质量问题→卖家承担；非质量问题→买家承担
退款时效：收到退货后3-5个工作日
```

### 退货处理流程
```
买家申请退货 → 审核原因 → 
  ├── 质量问题   → 直接退款(不退货) / 补发
  ├── 尺寸不匹配  → 换货 / 退款
  └── 不喜欢     → 按退货政策处理
```

### 减少退货的措施
1. 尺码表放详情页首位
2. 用视频展示真实效果
3. 写明材质/重量/颜色说明
4. 鼓励下单前咨询客服
5. 包裹内放使用指南

## 四、客户满意度提升

### 超预期体验
- 包裹内放手写感谢卡
- 随单送小礼品（贴纸/试用装）
- 复购客户专属折扣码
- 节日祝福短信/邮件

### 会员体系设计
```
普通会员：消费< $50
银卡会员：消费 $50-$200（5%折扣）
金卡会员：消费 $200-$500（10%折扣）
钻石会员：消费 > $500（15%折扣+优先发货）
```

### 复购激励
- 下次购买95折码（随包裹发送）
- 满3单减10%
- 生日月双倍积分
- 推荐朋友各得$5优惠券

## 五、客服团队管理

### 考核KPI
| 指标 | 目标 | 考核方式 |
|------|------|----------|
| 首次响应时间 | < 2小时 | 系统统计 |
| 解决率 | > 90% | 客户反馈 |
| 客户满意度 | > 4.5星 | 评价系统 |
| 退款率 | < 8% | 后台数据 |

### 常见场景话术库
建立20+常见场景的标准化回复，新人3天上手。
