# 跨境电商成本结构

## 成本全景图

```
┌─────────────────────────────────────────────────────┐
│                    售价 ¥100                         │
├─────────────────────────────────────────────────────┤
│ ■ 产品成本     ¥20 (20%)   采购价+包装               │
│ ■ 物流成本     ¥15 (15%)   头程+尾程+仓储             │
│ ■ 平台佣金     ¥5  (5%)    TikTok/Shopee等           │
│ ■ 广告营销     ¥15 (15%)   投放+达人                  │
│ ■ 支付与换汇   ¥3  (3%)   平台支付+跨境提现            │
│ ■ 退货损耗     ¥3  (3%)   退货折损                     │

└─────────────────────────────────────────────────────┘
```

## 各成本项详解

### 1. 产品成本
```
采购成本 = 产品出厂价 + 国内运费 + 质检 + 包装材料
```
| 供应链类型 | 典型采购倍率 | 说明 |
|------------|-------------|------|
| 1688 拿货 | 1× | 标准化产品 |
| OEM 定制 | 1.2-1.5× | 贴牌+包装定制 |
| ODM 开发 | 2-3× | 独家开模 |

### 2. 跨境物流成本
- **头程**：中国 → 海外仓，海运 $1-3/kg，空运 $3-8/kg
- **海外仓**：$0.5-2/件/月
- **尾程配送**：本土快递 $1-5/件

### 3. 平台佣金
| 平台 | 佣金率 | 额外费用 |
|------|--------|----------|
| TikTok Shop MY | 2%-4% | 交易手续费1%-2% |
| TikTok Shop TH | 3%-5% | - |
| Shopee | 2%-5% | 免运活动费 |
| Lazada | 2%-4% | 支付手续费 |

### 4. 广告成本
- TikTok Ads CPA 平均：$3-8（东南亚）
- 达人纯佣：15%-25%
- 坑位费：$50-500/场

### 5. 支付与换汇
- 跨境收款服务费：0.3%-1%
- 换汇汇损：1%-2.5%
- 提现手续费：免费或固定费用

### 6. 退货损失
```
退货成本 = 售价 × 退货率 × (1 - 残值率)
```
- 服装类退货率：10%-25%
- 电子类退货率：5%-10%
- 家居类退货率：3%-8%

### 7. 合规费用
- 印尼 BPOM 认证：$500-2000
- 马来西亚 SIRIM：$300-1000
- 欧盟 CE 认证：$500-5000
- 美国 FCC：$1000-5000
