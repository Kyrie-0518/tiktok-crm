# A-B测试方法论

## 一、A-B测试基础

### 什么是A-B测试
将用户随机分为A组和B组，展示不同版本，对比哪个效果更好。

```
┌─────────────┐    ┌─────────────┐
│   版本 A     │    │   版本 B     │
│ (对照组)     │ vs │ (实验组)     │
└─────────────┘    └─────────────┘
       ↓                  ↓
   用户行为数据     vs  用户行为数据
       ↓                  ↓
         ┌───────────────┐
         │  哪个更好？    │
         └───────────────┘
```

## 二、跨境电商A-B测试场景

### 产品页测试
| 变量 | 版本A | 版本B | 衡量指标 |
|------|-------|-------|----------|
| 主图 | 白底图 | 场景图 | CTR |
| 价格 | ¥99 | ¥89(划线¥129) | CVR |
| 标题 | 功能型 | 场景型 | CTR |
| 详情首屏 | 卖点列表 | 视频 | 停留时长 |

### 广告素材测试
- 钩子类型：惊讶vs痛点vs教程
- BGM：热门A vs 热门B
- 展示时长：15秒 vs 25秒
- 产品颜色/款式排列顺序

### 直播测试
- 开播时间：19点 vs 20点 vs 21点
- 主播话术：销售型 vs 分享型
- 福利款类型：低价品 vs 热门品

## 三、测试设计原则

### 一次只变一个变量
```
✅ 正确：
- 版本A：红色主图 + 场景型标题
- 版本B：蓝色主图 + 场景型标题
  （只改变主图颜色）

❌ 错误：
- 版本A：红色主图 + 低价
- 版本B：蓝色主图 + 高价
  （变了两个变量，不知道是哪个起了作用）
```

### 样本量要求
```
最小样本量 = 每组最少100次转化
理想样本量 = 每组500+次转化
测试时长 = 最少7天（覆盖完整周）
```

### 统计显著性
- p值 < 0.05 才是显著差异
- 注意：数据量不够不要下结论

## 四、测试执行SOP

### 准备阶段
1. 确定测试目标（提升什么指标）
2. 确定测试变量（只选一个）
3. 设计方案A和B
4. 预估所需样本量和时间

### 执行阶段
1. 设置测试（广告平台/落地页工具）
2. 流量均分（50:50）
3. 运行测试（至少7天不动）
4. 记录过程中出现的异常

### 分析阶段
1. 收集数据
2. 计算统计显著性
3. 分析结果
4. 输出测试报告

## 五、测试结果应用

### 结果判断
```
A显著优于B → 采用方案A，全量上线
B显著优于A → 采用方案B，全量上线
无显著差异 → 扩大样本再测 / 测试新变量
```

### 持续优化循环
```
提出假设 → 设计测试 → 执行测试 → 分析结果 → 应用结论 → 新假设 → ...
```

## 关键词

A-B测试、对照组、统计显著、转化优化、实验设计
