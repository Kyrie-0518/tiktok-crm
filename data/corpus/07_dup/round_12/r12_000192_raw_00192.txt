# TikTok 短视频与直播运营

## 一、短视频创作 SOP

### 前期准备
1. 确定选题：对标爆款 + 差异化
2. 写脚本：钩子 + 内容 + CTA
3. 准备道具：产品 + 背景 + 灯光
4. 场景布置：干净、明亮、简约

### 拍摄技巧
- **光线**：环形灯正面 + 侧光补光
- **角度**：多角度展示（正面/侧面/特写/对比）
- **稳定**：三脚架固定，避免晃动
- **时长**：15-30秒最佳

### 后期剪辑
1. 剪掉卡顿和冗余
2. 添加字幕（多语言）
3. 背景音乐调整
4. 加速无聊部分（1.2x-1.5x）
5. 添加文字动效

### 发布检查清单
- [ ] 视频无黑边/模糊
- [ ] 字幕无错别字
- [ ] BGM 合规（无版权问题）
- [ ] 标签 3-5 个
- [ ] 挂商品链接
- [ ] 评论区引导

## 二、直播运营全流程

### 开播前准备（Pre-Live）

**硬件清单**：
- 手机/摄像头（主+副2-3台）
- 环形灯 + 柔光箱
- 支架/云台
- 麦克风
- 背景布/展示台
- 提词器（可选）

**软件准备**：
- 测试网络（≥10Mbps上传）
- 测试商品链接
- 设置优惠券
- 准备背景音乐歌单

**内容准备**：
- 直播脚本（含话术）
- 产品排品顺序
- 互动环节设计（抽奖/问答）
- 应急话术（卡顿、冷场）

### 直播中运营（Live）
```
主播话术节奏：每5分钟一个产品循环
场控配合：上链接、发优惠券、回复评论
数据监控：在线人数、GPM、转化率
应急处理：卡顿切WiFi、冷场切互动
```

### 直播后复盘（Post-Live）

| 复盘维度 | 关注数据 | 优化方向 |
|----------|----------|----------|
| 流量 | 场观、峰值在线 | 预热视频/投流 |
| 转化 | GPM、转化率 | 话术/定价/节奏 |
| 内容 | 平均停留时长 | 增加互动环节 |
| 产品 | 单品点击率/转化率 | 排品顺序调整 |
| 粉丝 | 新增粉丝、粉丝GMV | 粉丝专属福利 |

## 三、提升直播间留存的技巧

1. **每10分钟一个钩子**：抽奖/福利款即将上架
2. **互动提问**：扣1/扣2，增加互动
3. **价格悬念**："这个价格你们绝对想不到"
4. **库存紧迫感**："只有最后20单"
5. **视觉变化**：切换镜头/换背景

## 四、常见问题与解决

| 问题 | 解决方案 |
|------|----------|
| 直播间没人 | 发预热视频 + 投流加热 |
| 在线但不下单 | 上福利款/加优惠力度/优化话术 |
| 卡顿掉线 | 备用网络/IronSocket |
| 退货率高 | 产品描述更真实/展示细节 |
| 差评 | 主动联系/补发/退款 |
