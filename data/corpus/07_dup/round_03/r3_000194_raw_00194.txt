# Facebook与Google广告投放

## 一、Facebook Ads投放指南

### 广告类型
| 类型 | 展示位置 | 适合 |
|------|----------|------|
| 图片广告 | Feed/Right Column | 引流 |
| 视频广告 | Feed/Stories/Reels | 种草 |
| 轮播广告 | Feed | 多产品展示 |
| 精品栏广告 | Feed | 商品目录 |
| 线索广告 | Feed | 收集用户信息 |

### Facebook受众定向
| 维度 | 选项 | 建议 |
|------|------|------|
| 地域 | 城市级定向 | 初期选核心城市 |
| 年龄 | 18-65+ | 按产品精准 |
| 兴趣 | 购物/美妆/科技等 | 2-3个兴趣叠加 |
| 行为 | 网购用户/高消费 | 优先选择 |
| 自定义受众 | 上传用户邮箱 | 再营销用 |
| 相似受众 | 基于种子用户 | 放量用 |

### Facebook像素设置
```
1. 创建Facebook Pixel
2. 安装到独立站/落地页
3. 设置转化事件：
   - ViewContent（浏览商品）
   - AddToCart（加购）
   - Purchase（购买）
4. 积累数据 → 用于再营销和相似受众
```

### Facebook广告预算建议
- 起步：$10-20/天/广告组
- 学习期：3-7天不要动
- 优化节奏：24-48小时观察后调整

## 二、Google Ads投放指南

### 广告类型
| 类型 | 展示位置 | 适合 |
|------|----------|------|
| **搜索广告** | 搜索结果页 | 精准需求 |
| **购物广告** | Google Shopping | 电商必备 |
| **展示广告** | 联盟网站 | 品牌曝光 |
| **视频广告** | YouTube | 种草教育 |
| **效果最大化** | 全渠道 | AI自动优化 |

### Google Shopping设置
```
1. 创建Google Merchant Center账号
2. 上传商品Feed（标题/价格/图片/ID）
3. 连接Google Ads
4. 创建购物广告系列
5. 设置出价策略
```

### Google Shopping Feed优化
- **标题**：核心关键词 + 品牌 + 属性（颜色/尺寸/材质）
- **描述**：详细参数 + 使用场景
- **图片**：纯白背景、无水印、高清
- **价格**：有竞争力
- **GTIN**（全球贸易代码）：有的话填入，提升匹配率

### 搜索广告关键词策略
- **广泛匹配**：覆盖更多搜索 → 但可能不精准
- **短语匹配**：中等精准
- **精确匹配**：最精准，点击最贵
- **否定关键词**：排除不相关的搜索

## 三、TikTok vs Facebook vs Google 对比

| 维度 | TikTok Ads | Facebook Ads | Google Ads |
|------|-----------|-------------|------------|
| 用户意图 | 被动发现 | 半被动 | 主动搜索 |
| 适合品类 | 视觉型/冲动 | 各类 | 需求明确型 |
| CPM | $1-8 | $3-12 | $5-20 |
| CPA | $3-15 | $5-20 | $8-30 |
| 转化周期 | 短(冲动) | 中 | 中长 |
| 学习曲线 | 较低 | 中 | 高 |

## 四、多渠道投放策略

### 预算分配建议
```
测试期（月预算$3000）：
- TikTok: 40%（测品快）
- Facebook: 30%（扩人群）
- Google: 30%（搜索意图）

稳定期（月预算$10000）：
- TikTok: 35%
- Facebook: 25%
- Google: 25%
- 备用/测试: 15%
```
