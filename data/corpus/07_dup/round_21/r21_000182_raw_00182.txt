# 跨境电商财务报表与分析

## 一、三张核心报表

### 1. 利润表（P&L）
```
┌────────────────────────────────────┐
│           月 度 利 润 表             │
├────────────────────────────────────┤
│ 总收入（GMV - 退款）       100,000  │
│ 减：产品成本                -25,000  │
│ 减：平台佣金                -5,000   │
│ 减：物流费                  -15,000  │
│ 减：广告费                  -12,000  │
│ 减：仓储费                  -3,000   │
│ 减：支付手续费              -1,000   │
│ 减：人员工资                -10,000  │
│ 减：其他费用                -2,000   │
├────────────────────────────────────┤
│ 净利润                      27,000  │
│ 净利率                      27%     │
└────────────────────────────────────┘
```

### 2. 现金流量表
关注：本月收款/付款/净现金流入
- 避免"账面盈利但现金流断裂"

### 3. 产品利润表
每个SKU单独核算利润，用于淘汰亏损品。

## 二、关键财务指标

| 指标 | 公式 | 健康线 |
|------|------|--------|
| 毛利率 | (收入-产品成本)/收入 | ≥ 60% |
| 净利率 | 净利润/收入 | ≥ 15% |
| 广告费比 | 广告费/收入 | ≤ 15% |
| 物流费比 | 物流费/收入 | ≤ 15% |
| 退货率 | 退货金额/GMV | ≤ 10% |
| ROI | 净利润/总投入 | ≥ 30% |

## 三、财务分析四步法

### Step 1：收入分析
- 哪个市场贡献最大？
- 哪个平台ROI最高？
- 日销/周销趋势？

### Step 2：成本分析
- 最大成本项是什么？
- 哪项成本在增长？
- 与上月对比？

### Step 3：利润分析
- 净利率趋势（月度）
- 哪些产品在亏损？
- 利润构成是否健康？

### Step 4：现金流分析
- 本月净现金流？
- 应收账款？
- 是否有资金压力？

## 四、财务预警指标

| 预警信号 | 危险值 | 行动计划 |
|----------|--------|----------|
| 净利率连续下降 | 连续3月↓ | 排查成本项 |
| 广告费比上升 | > 20% | 关停高CPA广告 |
| 退货率飙升 | 单月 > 15% | 检查产品质量 |
| 现金流负 | 连续2月 | 收缩投入 |
| 单个SKU亏损 | > 30天 | 立即停售 |
